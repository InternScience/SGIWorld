Key Insights from Evaluations

Multimodal Breaking 60

Gemini 3 Pro leads Scientific Multimodal Capability with 62.88, reflecting strong performance in perception and reasoning.

Open-Source Catching Up

Qwen3-VL-235B and Qwen3-Max now match or surpass several proprietary models in symbolic reasoning.

Code Generation Bottleneck

No model exceeds 30 in end-to-end executable code tasks, indicating scientific workflows still fail frequently.

Symbolic Reasoning Gap

No model exceeds 50 in equation-level manipulation, revealing fundamental limitations.

Quick Start

Get from clone to first scores in minutes

# Install
git clone https://github.com/InternScience/SciEvalKit.git
cd SciEvalKit
pip install -e .[all]

# Run evaluation
python run.py \
  --dataset SFE \
  --model gpt-4o \
  --mode all \
  --work-dir outputs/demo
